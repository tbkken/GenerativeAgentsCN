## 引用原则

1. 优先引用论文、官方仓库、官方文档。
2. 技术事实必须能回到来源。
3. 项目实现必须使用本地源码作为证据。
4. 不使用二手博客替代论文。
5. 不大段复制论文原文。
6. 前沿章节必须明确“当前项目已有能力”和“建议升级能力”的区别。
